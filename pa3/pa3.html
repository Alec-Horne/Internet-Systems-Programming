Programming Assignment 3

This webpage is an example of a simple "web page generator". It features 22 different elements in HTML, CSS, and Javascript that can be added to a new page that will be opened when the "show window" button is clicked. All of the features that are available are listed at the bottom of the page in a table with descriptions explaining each of them.

Directions: First click the show window button. The window must be open before elements can be added. After the window is open any element that is added will be dynamically placed on the new window. No need to click show window afterwards. You can watch the elements being placed. Certain browsers wont display certain elements.

Elements List

Add Color Flashing Text Adds color flashing text to the window

Add embedded Video Adds an embedded video from youtube in a small frame to the window
Add Image Adds the image in the textbox or a random image from a set group of images to the window

Add Addition Calculator Adds a simple addition calculator that takes two integers from user and displays the result of the addition
Add Responsive Slider Bar Adds a responsive range slider bar to the window
Add Animated Progress Bar Adds an animated progress bar that loops infinitely from 0% to 100%
